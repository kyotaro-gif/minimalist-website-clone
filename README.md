# Minimalist Website Clone

This project is a responsive minimalist website clone built using pure HTML and CSS.

## Features
- Clean and modern UI
- Fully responsive design
- Flexbox and CSS Grid layout
- Smooth hover effects
- Mobile-first approach

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid

## Purpose
This project was built to practice responsive layouts and modern CSS techniques.
